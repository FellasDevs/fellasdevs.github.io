Esse projeto foi feito utilizando as principais tecnologias:
No back-end:
  • A linguagem Go
  • O framework web Gin
  • O banco de dados PostgreSQL
  • O ORM Gorm
  • Json Web Tokens
No front-end:
  • A linguagem Typescript
  • O framework web Next.js
  • O framework de estilização TailwindCSS
  • A biblioteca de gerenciamento de formulários React-Hook-Form
  • A biblioteca de componentes Shadcn
  • A biblioteca de queries React-Query
  • A biblioteca de requisições Axios
  • A biblioteca de validações de formulário Zod
